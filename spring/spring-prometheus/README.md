## Prometheus Setup
1. Download prometheus and update prometheus.yml as below:
2. Start Prometheus.
```
scrape_configs:      
      - job_name: local_spring_boot_metric
        scrape_interval: 3s
        metrics_path: /actuator/prometheus
        static_configs:
          - targets: ["localhost:8080"]
```
